# Tic-Tac-Toe Game (Java GUI)

**Name:** JoyJumbo
**Student ID:** 20220793


## Description
A beginner-friendly 2-player Tic-Tac-Toe game built in Java using Swing. Players take turns, and the game announces a winner or a draw.

## Compilation and Execution
javac com/tictactoe/model/*.java com/tictactoe/controller/*.java com/tictactoe/view/*.java
java -cp . com.tictactoe.view.TicTacToe


## OOP Principles Used
- **Encapsulation**: Private fields with public getters.
- **Association**: Game uses Board and Player objects.
- **Separation of Concerns**: Model, Controller, View are split into packages.
